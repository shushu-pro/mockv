## 日志规范

### 版本格式

版本号（Major.Minor.Patch） 时间（yyyy/MM/dd）

### 描述类型

- 新增 ✨（Features）：新增功能。
- 修复 🐛（Fixed）：修复 bug。
- 更新功能 🕹 （Changed）：更新 API。
- 更新 UI 💄（Ui）：代码 UI 更新
- 更新样式 🎨 （Styles）：改进结构和代码格式。
- 优化 ⚡（Refactored）：性能优化，并未带来功能的逻辑变化。
- 废弃 ⚠（Deprecated）：不建议使用 / 在以后的版本中即将删除的功能。
- 删除 💥（Removed）：已删除的功能。
- 文档 📝（Doc）：文档编辑。
- 测试 ✅（Test）： 添加测试用例
- 启动 🎉（Init）： 项目初始化
